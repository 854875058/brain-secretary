# memory/ 目录说明

这里存放更细粒度的长期记忆条目（`*.md`），供 OpenClaw 的 memory 工具检索：

- `MEMORY.md`：入口与索引（放“稳定事实/关键决策/约定”）
- `memory/*.md`：按主题拆分（例如 `user.md`、`projects.md`、`decisions.md`、`todos.md`）
- `memory/qq-memory-ledger.jsonl`：桥接层自动沉淀的结构化记忆流水
- `memory/work-rules.md` / `memory/user-preferences.md` / `memory/workflow-patterns.md`：桥接层按主题沉淀的长期记忆

建议写法：

- 每条记录尽量短小
- 重要信息包含明确日期（例如 2026-03-06）
- 避免写入密钥/token 等敏感信息
